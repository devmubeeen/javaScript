Targeting DOM Element with ID, Class & Tag

  • Select by Id:
    $('#idName')
  • Select by Class Name:
    $('.className')
  • Select by Tag Name:
    $('tagName') or $("p")

Advance Selector

  • Select all element:
    $('*')
  • Select child element:
    $('ul li')
  • Select multiple classes
    $('.abc, .xyz')
  • Select multiple element
    $('h1, div, p')
  • Select in page first p element
    $('p:first')
  • Select in page last p element
    $('p:last')
  • Select li even element
    $('li:even')
  • Select li odd element
    $('li:odd')
